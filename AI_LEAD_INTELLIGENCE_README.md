## ✨ Features

### 1. **AI Lead Scoring (0-100)**
Every lead gets an intelligent score based on:
- Message quality and detail level
- Urgency indicators (ASAP, urgent, specific timelines)
- Budget/financing mentions
- Specific requirements and questions
- Contact information completeness
- Project specificity

**Score Categories:**
- 🔥 **HOT (80-100)**: Detailed inquiry, clear intent, immediate follow-up needed
- ⭐ **WARM (50-79)**: Good details, some urgency, worth pursuing
- ❄️ **COLD (0-49)**: Generic inquiry, needs nurturing

### 2. **Lead Intent Analysis**
AI extracts key information from lead messages:
- **Budget Range**: Extracted from mentions like "$500k-700k"
- **Timeline**: When they want to move/buy
- **Priorities**: What features matter most (parking, bedrooms, location, etc.)
- **Buyer Type**: Investor, End-user, First-time buyer, Upgrader, or Agent
- **Urgency Level**: High, Medium, or Low

### 3. **Smart Project Matching**
For each lead, AI finds and ranks 5 similar projects to pitch based on:
- Same city/location
- Similar price range (±20%)
- Matching bedroom count
- Same builder or similar reputation
- Similar features and amenities

### 4. **AI-Generated Pitches**
For each recommended project:
- **Match Score**: Percentage match (95%, 88%, etc.)
- **Reasoning**: Why this project fits the lead
- **Pitch**: One-sentence compelling pitch you can use
- **Key Differentiators**: 3-4 bullet points highlighting unique features

## 📊 What Each Score Means

### 🔥 HOT LEAD (80-100)
**Characteristics:**
- Detailed, specific inquiry
- Clear budget and timeline mentioned
- Specific requirements stated
- Complete contact information
- High urgency indicators

**Action:** Call immediately, highest priority

**Example:**
> "Hi, I'm looking for a 2-bedroom condo in Mississauga, budget around $600k. Need parking and close to transit. Looking to move by June. Can you show me options this week?"

---

### ⭐ WARM LEAD (50-79)
**Characteristics:**
- Good level of detail
- Some specifics provided
- Timeline may be mentioned
- Reasonable contact info
- Medium urgency

**Action:** Follow up within 24 hours

**Example:**
> "Interested in pre-construction condos in downtown Toronto. Budget is flexible. Looking for something with good amenities."

---

### ❄️ COLD LEAD (0-49)
**Characteristics:**
- Generic inquiry
- Minimal details
- No timeline or budget mentioned
- Incomplete contact info
- Low urgency

**Action:** Send automated email, follow up if they respond

**Example:**
> "Hi, tell me about this project."

## 🎓 Understanding the AI Insights

### Budget Detection
AI looks for phrases like:
- "$500k to $700k"
- "Around 600 thousand"
- "Budget is 1.2M"
- "Can afford up to..."

### Timeline Detection
AI looks for:
- Specific dates: "Moving in June 2025"
- Relative time: "Looking to buy in 6 months"
- Urgency: "ASAP", "Immediately", "This quarter"

### Priority Detection
AI identifies mentions of:
- Location preferences
- Number of bedrooms/bathrooms
- Parking requirements
- Transit access
- Amenities (gym, pool, etc.)
- Views, balconies, etc.

### Buyer Type Classification
- **Investor**: Mentions ROI, rental income, investment
- **End-user**: Mentions family, living, lifestyle
- **First-time buyer**: Mentions "first home", financing questions
- **Upgrader**: Mentions selling current home, upgrading
- **Agent**: Explicitly states they're an agent

## 🛠️ Technical Details

### API Endpoints

**1. Lead Analysis**
```
POST /api/leads/analyze
Body: { lead: <Lead object> }
Response: { analysis: { score, confidence, insights, reasoning, keyPoints } }
```

**2. Project Recommendations**
```
POST /api/leads/recommend-projects
Body: { lead: <Lead object>, originalProject?: <Project object> }
Response: { recommendations: [{ project, matchScore, reasoning, pitch, keyDifferentiators }] }
```

### AI Model
- **Provider**: DeepSeek AI
- **Model**: deepseek-chat
- **API Endpoint**: https://api.deepseek.com/v1/chat/completions
- **Temperature**: 0.3 (analysis), 0.5 (recommendations)

### Data Sources
- **Leads**: `fj_leads` and `precon_factory_leads` tables
- **Projects**: `canada_properties` table (3,224+ projects)
- **Matching**: SQL queries + AI ranking

## ❓ Troubleshooting

### "Analyzing Lead..." stuck loading
- Check your internet connection
- Verify DeepSeek API key is set in environment variables
- Check browser console for errors

### No recommendations showing
- Lead may not have enough project context
- No similar projects in database for that area/criteria
- Try clicking again to retry

### Analysis seems inaccurate
- AI works best with detailed lead messages
- Generic messages (like "tell me more") will score low
- The more context in the lead message, the better the analysis

### Match scores seem off
- AI considers multiple factors beyond just location
- Lower match scores may still be good opportunities
- Use the reasoning to understand why AI matched each project
