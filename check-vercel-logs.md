# Check Vercel Logs

1. Go to https://vercel.com/fahadjaveds-projects/property-dashboard/logs
2. Filter by last 10 minutes
3. Look for `/api/bookings/notify`
4. Look for `/api/bookings/create-calendar-event`
5. Copy any error messages you see

Common errors to look for:
- "Calendar not authorized"
- "Failed to create calendar event"
- "Token not found"
- Network errors
